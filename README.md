# online_store

Создайте папку для своего проекта, откройте его в VS Code;
В VS Code откройте терминал, вставьте туда ссылку, командой: git clone <путь репозитория>; (путь - зеленая кнопка вверху -> тык, тык);
Вам необходимо переместиться в папку с проектом test_store (для перемещения: cd <директория>)
Установите Python 3.10, в VS Code нажмите комбинацию Ctrl + Shift + P и в поиске вводим Python: Select interpreter, выбираем установленную версию;
Разверните виртуальное окружение командой: python -m venv venv; Включите его, venv\Scripts\activate; #Для отключения: deactivate
В терминале установите зависимости: pip install -r .\requirements.txt;
Возможно понадобится открыть порт 5000 в брандмауэре, как сделать - инстукция в инете;
В терминале запустите run.bat, если на винде и run.sh если на unix системе;
Когда проект запустится, появится порт, удерживая Ctrl нажмите на нее. Всё.
