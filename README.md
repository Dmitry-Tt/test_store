# online_store

1) Создайте папку для своего проекта, откройте его в VS Code;  
2) В VS Code откройте терминал, вставьте туда ссылку, командой: git clone <путь репозитория>; (путь - зеленая кнопка вверху -> тык, тык);  
3) Вам необходимо переместиться в папку с проектом test_store (для перемещения: cd <директория>)  
4) Установите Python 3.10, в VS Code нажмите комбинацию Ctrl + Shift + P и в поиске вводим Python: Select interpreter, выбираем установленную версию;  
5) Разверните виртуальное окружение командой: python -m venv venv; Включите его, venv\Scripts\activate; #Для отключения: deactivate  
6) В терминале установите зависимости: pip install -r .\requirements.txt;  
7) Возможно понадобится открыть порт 5000 в брандмауэре, как сделать - инстукция в инете;  
8) В терминале запустите run.bat, если на винде и run.sh если на unix системе;  
9) Когда проект запустится, появится порт, удерживая Ctrl нажмите на нее. Всё.
